# Répartition des tâches du soir

**Comment utiliser le programme:**

* Dans la boîte **Employés**, insérer le nombre de participants aux tâches.
* Dans la boîte **Tâches**, insérer le nombre de tâches à réaliser.

*Note : Pour que le programme fonctionne, il faut avoir un nombre égaux d'**Employés** et de **Tâches**. Un message d'erreur va apparaitre dans ce cas indiquant la nature de l'erreur.*
